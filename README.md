https://www.late-hack.com/
# サービス概要
全ての遅刻に悩む人を救うアプリです。<br>
以下の２つの機能で遅刻してしまう人を救います！
- 遅刻した理由を提示し、オリジナル遅延証明をする
- 待ち合わせ相手に怒られないためのテクニックを紹介する
